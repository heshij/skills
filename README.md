# Anthony Fu's Skills

A curated collection of [agent skills](https://agentskills.io/home) for Vue ecosystem tools and frameworks with Anthony Fu's choices/tastes.

## Installation

```bash
npx skills add antfu/skills
```

Learn more about the cli usage at [skills](https://github.com/vercel-labs/skills).

## Skills

When installing `antfu/skills`, all the following skills will be included.

### Generated Skills

Skills generated from source documentation, and tuned by Anthony.

| Skill | Description | Source |
|-------|-------------|--------|
| [vue](skills/vue/SKILL.md) | Vue.js core - reactivity, components, composition API | [vuejs/docs](https://github.com/vuejs/docs) |
| [nuxt](skills/nuxt/SKILL.md) | Nuxt framework - file-based routing, server routes, modules | [nuxt/nuxt](https://github.com/nuxt/nuxt) |
| [vite](skills/vite/SKILL.md) | Vite build tool - config, plugins, SSR, library mode | [vitejs/vite](https://github.com/vitejs/vite) |
| [unocss](skills/unocss/SKILL.md) | UnoCSS - atomic CSS engine, presets, transformers | [unocss/unocss](https://github.com/unocss/unocss) |

### Vendored Skills

Skills synced from vendor repositories for easier installation.

| Skill | Description | Source |
|-------|-------------|--------|
| [slidev](skills/slidev/SKILL.md) | Slidev - presentation slides for developers | [slidevjs/slidev](https://github.com/slidevjs/slidev) (Official) |
| [vueuse](skills/vueuse/SKILL.md) | VueUse - 200+ Vue composition utilities | [vueuse/skills](https://github.com/vueuse/skills) (Official) |
| [vue-best-practices](skills/vue-best-practices/SKILL.md) | Vue 3 + TypeScript best practices for Volar | [hyf0/vue-skills](https://github.com/hyf0/vue-skills) |
